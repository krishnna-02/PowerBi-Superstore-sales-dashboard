# 📊 Superstore Sales Performance Dashboard – Power BI

## 📌 Project Overview

This project presents a **Superstore Sales Performance Dashboard** developed using **Microsoft Power BI**. The dashboard provides valuable insights into sales performance, profitability, product categories, and regional performance through interactive visualizations and KPI indicators.

The primary objective of this dashboard is to help businesses monitor sales trends, evaluate profitability, and make data-driven decisions.

---

## 🎯 Project Objectives

* Analyze overall sales performance.
* Monitor profit generation across different regions.
* Evaluate category-wise sales contribution.
* Track product quantity sold.
* Measure discount impact on sales.
* Enable interactive filtering for deeper business analysis.

## 🧮 DAX Measures Used

DAX
Total Sales = SUM(Superstore[Sales])

Total Profit = SUM(Superstore[Profit])

Total Quantity = SUM(Superstore[Quantity])

Average Discount = AVERAGE(Superstore[Discount])

### Why Measures?

* Dynamic calculations
* Responsive to filters and slicers
* Better dashboard performance
* Follow Power BI best practices
